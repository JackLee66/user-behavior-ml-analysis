# User Behavior Analysis and Consumption Prediction

## Overview
This project analyzes user behavior data and predicts consumption tendencies using machine learning.

## Tech Stack
- Python
- pandas, numpy
- scikit-learn
- matplotlib

## Method
- Feature engineering from user behavior (views, likes, comments, shares)
- Logistic regression model
- Train-test split for evaluation

## Result
The model achieves stable performance in predicting user consumption tendencies.

## Run
pip install -r requirements.txt  
python main.py
